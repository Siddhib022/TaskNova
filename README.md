# TaskNova – Task Management App

TaskNova is a simple and efficient task management web application built using React.  
It allows users to create, manage, and organize daily tasks in a clean and responsive interface.

---

## 🚀 Live Demo
https://siddhib022.github.io/TaskNova/

---

## 📌 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive user interface
- Fast and dynamic UI using React
- Simple and clean design

---

## 🛠️ Tech Stack

- React
- JavaScript
- HTML5
- CSS3
- Vite

---
## ⚙️ Installation and Setup

1️⃣ Clone the repository:  
git clone https://github.com/Siddhib022/TaskNova.git

2️⃣ Navigate to the project folder: 
cd TaskNova

3️⃣ Install dependencies: 
npm install

4️⃣ Run the development server: 
npm run dev

The app will run on:
http://localhost:5173
